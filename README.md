## Introduction

This tic-tac-toe program is fully written in C as a practice for my C
programming skills. It is practically equivalent to the [C++ version](https://github.com/weehs-lab/tictactoe).

## How to Use

1. It will show up with player A first.
2. Write your position in (x,y).
3. It will repeat step 1 and 2 but for player B.
4. The game continues until draw or either player A or B wins.

## Compilation

```bash
git clone https://github.com/weehs-lab/tictactoe-c.git
cd ./tictactoe-c
make all
./tictactoe-c
```
## Notes

This project is written for Debian 13. It should be working on all environments
as this project is written with C standard libraries.
